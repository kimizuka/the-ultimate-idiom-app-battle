# かんようくクエスト

小学生(中学年〜)向けの慣用句学習アプリ。`index.html` をブラウザで開くだけで動きます(オフライン可・インストール不要)。

## 企画概要

- **ねらい**: 慣用句を「暗記」ではなく「カード遊び・クイズ・スタンプ集め」で自然に覚えてもらう
- **対象**: 小学3〜6年生(全漢字にふりがな付き)
- **収録**: 慣用句50語
  - からだ(頭・目・耳・口・手・足など): 26語
  - どうぶつ(猫・馬・すずめ・虫など): 12語
  - くらし(油を売る・水に流すなど): 12語
  - 各語に「やさしい意味」と「学校・家庭が舞台の例文」を用意

## 機能

| モード | 内容 |
| --- | --- |
| ホーム | きょうの慣用句(日替わり)、がんばりメーター、称号表示 |
| おぼえる | カテゴリ別フラッシュカード。めくって意味を確認、「まだまだ」は後でもう一度出題 |
| クイズ | 全5問。意味あて4択+穴うめ(体の部位・動物を当てる)をランダム出題 |
| ずかん | 全50語の一覧(おぼえた語は⭐)、クイズのスタンプ帳 |

## 続けたくなる仕掛け

- 正解すると朱色の**はんこ**が「ドン!」と押される演出+効果音(🔊ボタンでオフ可)
- クイズ全問正解で**はなまる**スタンプ、スタンプ帳にたまっていく
- おぼえた語数で称号が上がる: 見習い → ことば探検家 → 名人 → 達人 → かんようく博士
- 進捗は端末の localStorage に自動保存

## 技術

- 単一HTMLファイル(依存ライブラリなし・外部通信なし)
- ふりがなは `{漢字|よみ}` 記法のデータを `<ruby>` に変換して表示
- ライト/ダークテーマ両対応、`prefers-reduced-motion` 対応
- 効果音は WebAudio で生成(音源ファイル不要)
